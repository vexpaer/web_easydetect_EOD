Diagnosis Model

The diagnosis model assesses an individual's risk of developing diseases. It uses CatBoost algorithm and 27 health-related features for prediction.

Risk Threshold: 0.53 - Values above indicate high risk, below indicate low risk.

Prognosis Model

The prognosis model predicts possible outcomes of disease progression. It uses Random Forest algorithm and 27 health-related features for prediction.

Risk Threshold: -12.03 - Values above indicate poor prognosis, below indicate good prognosis.
